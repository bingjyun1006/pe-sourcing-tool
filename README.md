# PE Sourcing Tool

私募股權供應鏈情報工具。透過 AI 自動從大廠年報與合併財報中萃取未上市供應鏈公司，建立可持續追蹤的賽道知識庫。

---

## 這個工具解決什麼問題

PE 盡職調查的核心難題之一，是在公開資訊有限的情況下找到值得投資的未上市供應鏈公司。

傳統方式依賴人工閱讀財報、整理附表，耗時且難以系統化。本工具讓 AI 接手這個過程：上傳一份財報 PDF，幾分鐘內自動輸出結構化的潛在標的清單。

---

## 核心功能

| 功能 | 說明 |
|------|------|
| **T1 年報軌道** | 上傳大廠年度報告，從採購揭露條款萃取供應商（含未上市） |
| **T2 財報軌道** | 上傳合併財務報告，從附表九／附表六萃取被投資未上市公司 |
| **供應鏈骨架地圖** | AI 動態生成 L1–L4 供應鏈層級結構，標示已知大廠位置 |
| **T1/T2 建議清單** | AI 推薦值得上傳分析的大廠名單，附上推薦理由 |
| **公司簡介查詢** | 點擊任一潛在標的，即時查詢業務概述、成立年份、主要客戶 |
| **書籤與追蹤清單** | 對感興趣的公司加星號，一鍵匯出追蹤清單 CSV |
| **多賽道管理** | 可同時建立並切換多個賽道（如：散熱模組、先進封裝、被動元件） |

## 系統設計

```
使用者上傳 PDF
      │
      ▼
PyMuPDF 讀取全文
      │
      ▼
Gemini 2.5 Flash 萃取
  ├─ T1：供應商名稱 + 業務描述
  └─ T2：被投資公司 + 持股比例
      │
      ▼
TWSE 資料庫比對上市狀態
      │
      ▼
過濾境外公司／雜訊
      │
      ▼
寫入賽道 Markdown 檔案
      │
      ▼
Streamlit UI 顯示潛在目標清單
```

**技術選型原因**
- **Gemini 2.5 Flash**：1M context window，可一次讀入完整財報，不需分段處理；免費額度足夠 MVP 使用
- **Streamlit**：Python 原生 UI 框架，無需前端開發，適合快速 MVP
- **Markdown 儲存**：結果存為純文字，易讀、易轉移、不依賴資料庫

## 使用方式

### 前置條件
- 取得 Gemini API Key：前往 [aistudio.google.com/apikey](https://aistudio.google.com/apikey) 免費申請

### 操作流程

1. 開啟工具網址，在頂端輸入 Gemini API Key
2. 在搜尋欄輸入賽道關鍵字（例：散熱模組、先進封裝）
3. 點擊「◈ 分析賽道」生成供應鏈骨架地圖與建議上傳清單
4. 依建議清單上傳對應公司的年報（T1）或合併財報（T2）PDF
5. 點擊「▶ 開始分析」，等待 AI 萃取完成
6. 在「已發現潛在目標」區塊查看結果，點擊公司名稱查詢詳細簡介
7. 對感興趣的公司加星號，完成後匯出追蹤清單 CSV

### 上傳文件選擇原則

| 文件類型 | 上傳至 | 預期產出 |
|----------|--------|----------|
| 年度報告（Annual Report） | T1 軌道 | 供應商名稱（多數大廠以代號取代，0 筆屬正常） |
| 合併財務報告（含附表九） | T2 軌道 | 被投資公司清單，準確率較高 |

> **建議優先上傳合併財報（T2）**，資訊揭露較完整，萃取成功率高。

## 資料說明

- 所有萃取結果儲存在 `Obsidian_Vault/01_Fact_Hunter/track_trees/` 目錄下
- 搜尋分析快取儲存在 `data/searches/`
- 上市狀態資料來源：TWSE 公開資料，每次啟動時即時比對
- AI 萃取資料僅供參考，上市狀態與業務描述建議再次核實

## 本地端啟動

```bash
python -m streamlit run app.py --server.port 8502
```
